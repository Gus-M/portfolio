Este trabalho foi sugerido pela Data Science Academy durante o curso 'Formao Cientista de Dados'. Os dados utilizados esto disponveis na competio Santander Customer Satisfation

Introdução¶

Para qualquer empresa, o nvel de satisfao do cliente uma das principais mtricas de sucesso. Clientes insatisfeitos no costumam permanecer em contato por muito tempo e tendem a buscar solues para seus problemas nos concorrentes.

Se identificarmos antecipadamente quais clientes esto insatisfeitos com os servios prestados pela empresa, poderemos aplicar tcnicas direcionadas a esse grupo para compreender suas frustraes, buscar feedback e possivelmente convert-los ao grupo de clientes satisfeitos, mantendo-os em contato com a empresa por mais tempo.

O Banco Santander j realiza este trabalho. Eles disponibilizaram centenas de variveis annimas sobre seus clientes e lanaram o desafio: identificar quais clientes esto satisfeitos ou insatisfeitos com os servios prestados pelo banco.

Definindo o problema de negócio¶

Objetivo: Identificar quais clientes esto satisfeitos e insatisfeitos com base nos dados disponveis

Meta: Desenvolver um modelo preditivo que consiga obter pelo menos 70% de acurcia.

Observaes: Todas variveis foram anonimizadas de forma que no seja possveis identificar informaes reais sobre os clientes do banco.

#Automatizamos o processo com Pipeline e ColumnTransformer
col_prep = ColumnTransformer(transformers=[('Standardize', StandardScaler(), colnames[:-1])])

def exec_pipe(model, X, y):
    model_pipe = Pipeline(steps=[('Prep', col_prep),
                                 ('Model', model)])
    return model_pipe.fit(X, y)
